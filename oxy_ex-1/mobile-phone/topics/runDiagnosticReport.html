Running problem diagnostic report

If you have connectivity problems, you can run a diagnostic report that helps you to identity the issues the phone might have. To run the report, follow this procedure:
  1. Select Menu > Settings.
  2. Choose Advanced Settings > Problem diagnostic.
  3. Press Run.
  4. Look for out of range data.
